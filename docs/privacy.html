Privacy Policy – Movie Scout

Movie Scout is an open-source application that allows users to discover, search for, and manage lists of movies and TV shows. This application uses the The Movie Database (TMDb) API to provide data and functionality.

Data Collection

Movie Scout does not collect or store any personal data. Some features, such as managing your watchlist or submitting ratings, require signing in with a TMDb account. However, this authentication is handled entirely by TMDb, and Movie Scout does not access or store any login credentials or personal user data.

Third-Party Services

Movie Scout uses The Movie Database (TMDb) API to retrieve movie and TV show data, as well as to manage user-related actions such as favorites, ratings, or lists. All user-related operations are processed directly by TMDb.
For more details, please review TMDb’s Privacy Policy.

Open Source

Movie Scout is fully open source. You can inspect, contribute to, or fork the source code at: GitHub Repository.

Contact

If you have any questions or concerns regarding privacy, please open an issue on the GitHub repository or contact the maintainers directly.
